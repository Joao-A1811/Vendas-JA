| Campo | Valor |
|---|---|
| Formato | E-book (arquivo PDF) |
| Categoria | Saúde e esportes / Finanças / Desenvolvimento pessoal (ver por produto) |
| Preço Brasil | **R$ 47,00** |
| Preço internacional (EN/ES) | **US$ 19** (criar oferta em dólar pro mesmo produto) |

## Depois de cadastrar cada produto

1. Copiar o **link de checkout** de cada oferta (Hotmart → Produto → Ofertas → link).
2. Mandar pro Claude (ou editar direto): o link BRL vai no `linkCheckoutHotmart` do
   `produtos/<slug>/index.html`, o link USD vai nas versões `-en` e `-es`.
3. Quando o produto estiver com PDF entregável + checkout funcionando: trocar
   `disponivel: false` → `true` no `assets/produtos.js`.
